# About chatgpt_to_claude

`chatgpt_to_claude` is a toolkit for reviewing and migrating ChatGPT export archives into Claude-friendly bundles.

It provides command-line tools and a small TUI to:

- parse common ChatGPT export formats
- generate Markdown previews and topic bundles
- extract and summarise attachments
- produce migration reports and traceable memory provenance

Current release: `v0.0.2`

See `README.md` for installation, usage, tests, and release instructions.
